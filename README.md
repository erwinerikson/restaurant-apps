# Katalog Restoran PWA
![Screenshot Aplikasi](halaman_web.jpg)

Aplikasi front-end web yang responsif, memiliki aksesibilitas yang baik, mudah di-maintenance, memiliki sifat native, dapat diuji, dan memiliki performa yang baik.

Website yang responsif menggunakan CSS Grid System, Media Query, dan Offscreen Canvas melalui pendekatan Mobile First.

Penerapan testing seperti Unit Test dan Integration Test pada website yang dibuat serta penggunaan DevTools untuk debugging.

Menggunakan web auditor untuk mengetahui sejauh mana performa website yang dibuat dan mengoptimalkan performa website berdasarkan hasil yang diperoleh.

Menerapkan teknik Continuous Integration/Continuous Deployment untuk melakukan proses testing dan build secara otomatis sehingga aplikasi dapat diakses secara publik.
